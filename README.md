# RAG - LLM Project

A Retrieval-Augmented Generation (RAG) system powered by Large Language Models (LLMs) for the NHS.

## Overview

This project implements a RAG pipeline that combines document retrieval with LLM capabilities to provide context-aware responses for healthcare-related queries.

## Setup

### Prerequisites
- Python 3.8+
- pip

### Installation

1. Clone the repository
2. Create a virtual environment:
   ```bash
   python -m venv venv
   ```
3. Activate the virtual environment:
   - **Windows**: `venv\Scripts\activate`
   - **macOS/Linux**: `source venv/bin/activate`

4. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
